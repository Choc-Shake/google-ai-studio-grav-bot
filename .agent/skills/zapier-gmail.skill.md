# Zapier Gmail Skill

**Description:**
Handles requests related to Gmail via Zapier MCP.

**Tools Required:**
- `zapier__gmail_find_email`
- `zapier__gmail_send_email`
- `zapier__gmail_reply_to_email`
- `zapier__gmail_create_draft`
- `zapier__gmail_create_draft_reply`

**Model Preference:**
`OPENROUTER_MODEL` (Heavy text generation for drafting emails)
